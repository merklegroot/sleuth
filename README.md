## Sleuth (Working Title)

Sleuth is a mystery investigation game inspired by the classic DOS game **“Sleuth”**. The player takes on the role of a detective, exploring a location, interviewing suspects, and piecing together clues to identify the culprit before time runs out.

## Core Concept

- **Goal**: Determine **who** committed the crime, **how** they did it, and possibly **why**, based on limited information and deductive reasoning.
- **Loop**:
  - Explore rooms and locations.
  - Examine objects and environmental clues.
  - Talk to suspects and note inconsistencies.
  - Use deduction to eliminate impossible scenarios.
  - Accuse a suspect when you’re confident.

## Inspirations from the Classic DOS Game

- **Randomized cases** so each playthrough is different.
- **Simple, readable visuals** that prioritize clarity over realism.
- **Logic and deduction first**, rather than action or reflexes.
- **Limited time or turns** to solve the case, creating tension.

## Initial Scope

- **Single location** (for example, a mansion) with a small set of rooms.
- **Small cast of suspects**, each with a short profile and alibi.
- **Basic clue system** that supports at least one solvable mystery.
- **Keyboard-driven controls** using the Raylib-based client.
